List

Mutable ¬∑ Ordered ¬∑ Indexed
Dynamic array supporting indexing, slicing, and mutation. O(1) append, O(n) insert/remove at arbitrary positions.
‚ñ∂ append(item) Add item to end O(1) 
# Adds item to the end of the list
lst = [1, 2, 3]
lst.append(4)  # [1, 2, 3, 4]
‚ñ∂ extend(iterable) Add all elements from iterable O(k) 
lst = [1, 2]
lst.extend([3, 4, 5])  # [1, 2, 3, 4, 5]
lst.extend((6, 7))      # [1, 2, 3, 4, 5, 6, 7]
‚ñ∂ insert(index, item) Insert item at index O(n)

Shifts elements to the right to make room.

lst = [1, 2, 4]
lst.insert(2, 3)  # [1, 2, 3, 4]
lst.insert(0, 0)  # [0, 1, 2, 3, 4]
‚ñ∂ remove(item) Remove first occurrence O(n)

‚ö† Raises ValueError if item not found.

lst = [1, 2, 3, 2]
lst.remove(2)  # [1, 3, 2] ‚Äî only first 2 removed
‚ñ∂ pop(index=-1) Remove & return item at index O(1) last / O(n) 
lst = [1, 2, 3]
val = lst.pop()   # Returns 3, lst = [1, 2]
val = lst.pop(0) # Returns 1, lst = [2]
‚ñ∂ clear() Remove all items O(n) 
lst = [1, 2, 3]
lst.clear()  # []
‚ñ∂ index(item, start, end) Find index of first occurrence O(n)

‚ö† Raises ValueError if item not found.

lst = [1, 2, 3, 2]
idx = lst.index(2)     # 1
idx = lst.index(2, 2)  # 3 (search from index 2)
‚ñ∂ count(item) Count occurrences O(n) 
lst = [1, 2, 2, 3, 2]
cnt = lst.count(2)  # 3
‚ñ∂ sort(key=None, reverse=False) Sort in-place O(n log n) 
lst = [3, 1, 4, 1, 5]
lst.sort()                # [1, 1, 3, 4, 5]
lst.sort(reverse=True)   # [5, 4, 3, 1, 1]

lst = [('a', 3), ('b', 1), ('c', 2)]
lst.sort(key=lambda x: x[1])
# [('b', 1), ('c', 2), ('a', 3)]
‚ñ∂ reverse() Reverse in-place O(n) 
lst = [1, 2, 3]
lst.reverse()  # [3, 2, 1]
‚ñ∂ copy() Shallow copy O(n) 
lst1 = [1, 2, 3]
lst2 = lst1.copy()  # [1, 2, 3]
lst2.append(4)     # lst1 unchanged
‚ñ∂ Slicing [start:end:step] Get/set sub-sequence O(k) 
lst[1:3]   # Elements at index 1, 2
lst[::2]   # Every 2nd element
lst[::-1]  # Reversed copy
lst[1:] = [x, y, z]  # Replace from index 1
▶ List Comprehension Build lists with concise syntax O(n) 
# Basic comprehension
squares = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]

# With condition
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# Nested comprehension (2D)
matrix = [[0] * 3 for _ in range(3)]
flat = [x for row in matrix for x in row]
▶ in / not in Membership test O(n) 
lst = [1, 2, 3, 4]
3 in lst      # True
5 not in lst  # True
▶ + (concat) / * (repeat) Concatenation and repetition O(n+m) / O(n*k) 
lst1 = [1, 2] + [3, 4]  # [1, 2, 3, 4]
lst2 = [0] * 5             # [0, 0, 0, 0, 0]

# Caution: nested mutable objects share refs
bad = [[]] * 3   # 3 refs to SAME list!
good = [[] for _ in range(3)]  # 3 separate lists

Tuple

Immutable ¬∑ Ordered ¬∑ Hashable
Immutable sequence supporting indexing and slicing. O(1) access, O(n) search. Use when data shouldn't change and for hashable keys.
‚ñ∂ index(item, start, end) Find first occurrence index O(n) 
t = (1, 2, 3, 2)
idx = t.index(2)  # 1
‚ñ∂ count(item) Count occurrences O(n) 
t = (1, 2, 2, 3, 2)
cnt = t.count(2)  # 3
‚ñ∂ Unpacking & Slicing Destructure and slice tuples O(n) 
a, b, c = (1, 2, 3)
x, *rest = (1, 2, 3, 4)  # x=1, rest=[2,3,4]

t[1:3]   # (2, 3)
t[::-1]  # Reversed tuple
▶ + (concat) / * (repeat) Concatenation and repetition O(n+m) / O(n*k) 
t1 = (1, 2) + (3, 4)  # (1, 2, 3, 4)
t2 = (0,) * 5           # (0, 0, 0, 0, 0)
# Single element tuple requires trailing comma

len(t)         # 4  ‚Äî O(1)
3 in t         # True  ‚Äî O(n)

Set

Unordered ¬∑ Unique ¬∑ O(1) Lookup
Unordered collection of unique elements. O(1) average for add/remove/lookup. Automatically eliminates duplicates.
‚ñ∂add(elem)Add element (no effect if exists)O(1)
s = {1, 2, 3}
s.add(4)  # {1, 2, 3, 4}
s.add(3)  # {1, 2, 3, 4} ‚Äî no change
‚ñ∂remove(elem)Remove element (raises KeyError)O(1)

‚ö† Raises KeyError if element not present.

s = {1, 2, 3}
s.remove(2)  # {1, 3}
‚ñ∂discard(elem)Remove if present (no error)O(1)
s = {1, 2, 3}
s.discard(2)  # {1, 3}
s.discard(5)  # {1, 3} ‚Äî no error
‚ñ∂pop()Remove & return arbitrary elementO(1)
s = {1, 2, 3}
elem = s.pop()  # Could be any of 1, 2, 3
‚ñ∂union(*others) | All elements from both setsO(n+m)
s1 = {1, 2, 3}
s2 = {3, 4, 5}
s3 = s1.union(s2)  # {1, 2, 3, 4, 5}
s3 = s1 | s2      # Same result
‚ñ∂intersection(*others) &Elements common to allO(min)
s1 = {1, 2, 3}
s2 = {2, 3, 4}
s3 = s1 & s2  # {2, 3}
‚ñ∂difference(*others) -Elements in first but not othersO(n)
s1 = {1, 2, 3}
s2 = {2, 3, 4}
s3 = s1 - s2  # {1}
‚ñ∂symmetric_difference(other) ^Elements in either but not bothO(n+m)
s1 = {1, 2, 3}
s2 = {2, 3, 4}
s3 = s1 ^ s2  # {1, 4}
‚ñ∂issubset / issuperset / isdisjointSet relationship checksO(n)
s1 = {1, 2}; s2 = {1, 2, 3}
s1.issubset(s2)     # True  (s1 <= s2)
s2.issuperset(s1)   # True  (s2 >= s1)

s3 = {4, 5}
s1.isdisjoint(s3)   # True  (no common elements)
‚ñ∂In-place: update |= &= -= ^=Modify set in-placeO(n)
s = {1, 2}
s |= {3, 4}  # update: {1, 2, 3, 4}
s &= {2, 3}  # intersection_update: {2, 3}
s -= {3}     # difference_update: {2}
▶clear()Remove all elementsO(n)
s = {1, 2, 3}
s.clear()  # set()
▶copy()Shallow copy of the setO(n)
s1 = {1, 2, 3}
s2 = s1.copy()  # {1, 2, 3} ‚Äî independent copy
s2.add(4)       # s1 unchanged
▶frozenset(iterable)Immutable & hashable setO(n)
fs = frozenset([1, 2, 3])

# Can be used as dict key or set element
d = {fs: "value"}
s = {frozenset({1, 2}), frozenset({3, 4})}

# Supports all non-mutating set operations
fs1 = frozenset({1, 2}) | frozenset({3})  # frozenset({1, 2, 3})
▶Set ComprehensionBuild sets with concise syntaxO(n)
squares = {x**2 for x in range(5)}
# {0, 1, 4, 9, 16}

unique_lengths = {len(w) for w in ["hi", "hello", "hey"]}
# {2, 5, 3}

Dictionary

Key-Value ¬∑ Ordered (3.7+) ¬∑ O(1)
Key-value pairs, insertion-ordered since Python 3.7. O(1) average for lookup, insert, delete. Keys must be hashable.
‚ñ∂keys() / values() / items()View objects for iterationO(n) iter
d = {'a': 1, 'b': 2, 'c': 3}
d.keys()    # dict_keys(['a', 'b', 'c'])
d.values()  # dict_values([1, 2, 3])
d.items()   # dict_items([('a',1), ('b',2), ('c',3)])

for key, val in d.items():
    print(key, val)
‚ñ∂get(key, default=None)Safe access ‚Äî never raises KeyErrorO(1)
d = {'a': 1, 'b': 2}
d.get('a')      # 1
d.get('x')      # None
d.get('x', 0)   # 0 (custom default)
‚ñ∂pop(key, default)Remove key & return valueO(1)
d = {'a': 1, 'b': 2}
val = d.pop('a')       # 1, d = {'b': 2}
val = d.pop('x', None) # None (no error)
‚ñ∂popitem()Remove & return last pairO(1)
d = {'a': 1, 'b': 2, 'c': 3}
key, val = d.popitem()  # ('c', 3)
‚ñ∂update(mapping)Merge key-value pairsO(m)
d = {'a': 1, 'b': 2}
d.update({'b': 20, 'c': 3})
# {'a': 1, 'b': 20, 'c': 3}
d.update(x=10, y=20)  # keyword args
‚ñ∂setdefault(key, default)Get or create key with defaultO(1)
d = {'a': 1}
d.setdefault('a', 10)  # 1 (exists)
d.setdefault('b', 2)   # 2 (created)
‚ñ∂fromkeys(iterable, value)Create dict from keysO(n)
d = dict.fromkeys(['a', 'b', 'c'])
# {'a': None, 'b': None, 'c': None}
d = dict.fromkeys(range(3), 0)
# {0: 0, 1: 0, 2: 0}
‚ñ∂Direct Access: d[key], del, inBracket access, delete, membershipO(1)
d['key']          # Get (KeyError if missing)
d['key'] = value  # Set
del d['key']      # Delete (KeyError if missing)
'key' in d        # Check existence ‚Üí True/False
▶clear()Remove all key-value pairsO(n)
d = {'a': 1, 'b': 2}
d.clear()  # {}
▶copy()Shallow copy of the dictionaryO(n)
d1 = {'a': 1, 'b': [2, 3]}
d2 = d1.copy()
d2['c'] = 4  # d1 unchanged
# Warning: nested mutables are shared (shallow)
▶| merge / |= update (3.9+)Merge dictionaries with operatorsO(n+m)
d1 = {'a': 1, 'b': 2}
d2 = {'b': 20, 'c': 3}

d3 = d1 | d2   # {'a': 1, 'b': 20, 'c': 3} (new dict)
d1 |= d2        # in-place merge (like update)
▶Dict ComprehensionBuild dicts with concise syntaxO(n)
squares = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# Swap keys and values
d = {'a': 1, 'b': 2}
inv = {v: k for k, v in d.items()}  # {1: 'a', 2: 'b'}

# With condition
evens = {k: v for k, v in d.items() if v % 2 == 0}

Deque

collections ¬∑ Double-Ended Queue
from collections import deque ‚Äî Optimized for O(1) operations at both ends. Ideal for BFS and sliding window.
‚ñ∂append(x) / appendleft(x)Add to right / left endO(1)
from collections import deque
dq = deque([1, 2, 3])
dq.append(4)       # deque([1, 2, 3, 4])
dq.appendleft(0)   # deque([0, 1, 2, 3, 4])
‚ñ∂pop() / popleft()Remove from right / left endO(1)
dq = deque([1, 2, 3])
val = dq.pop()      # 3
val = dq.popleft()  # 1, dq = deque([2])
‚ñ∂extend(iter) / extendleft(iter)Add multiple elementsO(k)
dq = deque([1, 2])
dq.extend([3, 4])       # deque([1, 2, 3, 4])
dq.extendleft([5, 4, 3]) # deque([3, 4, 5, 1, 2, 3, 4])
# Note: extendleft reverses order
‚ñ∂rotate(n)Rotate right (positive) or left (negative)O(n)
dq = deque([1, 2, 3, 4])
dq.rotate(1)   # deque([4, 1, 2, 3])
dq.rotate(-2)  # deque([2, 3, 4, 1])
▶remove(val) / count(val) / index(val)Search and count elementsO(n)
dq = deque([1, 2, 3, 2, 4])
dq.remove(2)       # deque([1, 3, 2, 4]) ‚Äî first occurrence
dq.count(2)        # 1
dq.index(3)        # 1 ‚Äî first occurrence index
▶reverse() / clear() / copy() / insert()In-place reverse, clear, copy, insertO(n)
dq = deque([1, 2, 3])
dq.reverse()    # deque([3, 2, 1]) ‚Äî in-place
dq.insert(1, 9) # deque([3, 9, 2, 1]) ‚Äî O(n)
dq2 = dq.copy() # shallow copy
dq.clear()      # deque([])
▶deque(iterable, maxlen)Bounded deque ‚Äî auto-discards old itemsO(1)
# Fixed-size deque (great for sliding window)
dq = deque(maxlen=3)
dq.append(1); dq.append(2); dq.append(3)
# deque([1, 2, 3], maxlen=3)
dq.append(4)  # deque([2, 3, 4]) ‚Äî 1 dropped

dq.maxlen  # 3 (read-only attribute)

Counter

collections ¬∑ Frequency Counter
from collections import Counter ‚Äî Subclass of dict for counting hashable objects efficiently.
‚ñ∂most_common(n)Top n most common elementsO(n log n)
from collections import Counter
cnt = Counter([1, 2, 2, 3, 3, 3])
cnt.most_common(2)  # [(3, 3), (2, 2)]
cnt.most_common()   # All, sorted by count

cnt = Counter([1, 2, 2, 3, 3, 3])
list(cnt.elements())  # [1, 2, 2, 3, 3, 3]

cnt = Counter([1, 2, 2])
cnt.update([2, 3, 3, 3])  # {1:1, 2:3, 3:3}
cnt.subtract([2, 3])       # {1:1, 2:2, 3:2}

c1 = Counter(a=3, b=1)

c1 + c2  # Counter({a: 4, b: 3})
c1 - c2  # Counter({a: 2}) ‚Äî positive only
c1 & c2  # Counter({a: 1, b: 1}) ‚Äî min
c1 | c2  # Counter({a: 3, b: 2}) ‚Äî max
▶total() (3.10+)Sum of all countsO(n)
cnt = Counter(a=3, b=2, c=1)
cnt.total()  # 6 (sum of all counts)

# For Python < 3.10 use:
sum(cnt.values())  # 6

defaultdict

collections ¬∑ Auto-Default Values
from collections import defaultdict ‚Äî Dict that auto-creates default values for missing keys. All standard dict methods work.
‚ñ∂defaultdict(factory)Auto-initialize on missing key accessO(1)
from collections import defaultdict

dd = defaultdict(int)
dd['x'] += 1  # 'x' ‚Üí 0, then +1 = 1

dd = defaultdict(list)
dd['a'].append(1)  # 'a' ‚Üí [], then append

dd = defaultdict(lambda: 'N/A')
dd['missing']  # returns 'N/A'
▶default_factory / Nested defaultdictAccess factory & create nested structuresO(1)
dd = defaultdict(int)
dd.default_factory  # <class 'int'>
dd.default_factory = list  # change factory

# Nested defaultdict (adjacency list, 2D map)
graph = defaultdict(lambda: defaultdict(int))
graph['A']['B'] = 5  # auto-creates both levels

# Group by multiple keys
dd = defaultdict(lambda: defaultdict(list))

OrderedDict

collections ¬∑ Insertion Order
from collections import OrderedDict ‚Äî Dict remembering insertion order (regular dict also does in Python 3.7+). Useful for LRU Cache.
‚ñ∂move_to_end(key, last=True)Move key to end or beginningO(1)
from collections import OrderedDict
od = OrderedDict([('a',1), ('b',2), ('c',3)])
od.move_to_end('a')              # b, c, a
od.move_to_end('b', last=False)  # b, c, a
▶popitem(last=True)Pop from end (LIFO) or start (FIFO)O(1)
od = OrderedDict([('a',1), ('b',2), ('c',3)])
od.popitem()            # ('c', 3) ‚Äî last (default)
od.popitem(last=False)  # ('a', 1) ‚Äî first

# Unlike regular dict.popitem() which only pops last
▶Equality (order-sensitive)Order matters for == comparisonO(n)
od1 = OrderedDict([('a',1), ('b',2)])
od2 = OrderedDict([('b',2), ('a',1)])
od1 == od2  # False ‚Äî order differs

# Regular dicts ignore order:
{'a':1, 'b':2} == {'b':2, 'a':1}  # True

heapq

Min-Heap ¬∑ Priority Queue
import heapq ‚Äî Min-heap (binary heap). O(log n) push/pop, O(n) heapify. For max-heap, negate values.
‚ñ∂heappush / heappopAdd item / remove smallestO(log n)
import heapq
heap = []
heapq.heappush(heap, 5)
heapq.heappush(heap, 2)
heapq.heappush(heap, 8)  # [2, 5, 8]
val = heapq.heappop(heap) # 2
‚ñ∂heapify(list)Transform list into heap in-placeO(n)
arr = [3, 1, 4, 1, 5, 9, 2]
heapq.heapify(arr)  # [1, 1, 2, 3, 5, 9, 4]
‚ñ∂heappushpop / heapreplaceCombined push+pop operationsO(log n)
heap = [2, 5, 8]
# heappushpop: push first, then pop smallest
heapq.heappushpop(heap, 3)  # returns 2

# heapreplace: pop first, then push
heapq.heapreplace(heap, 1) # returns smallest, adds 1
‚ñ∂nlargest / nsmallestGet n largest/smallest elementsO(n log k)
arr = [1, 5, 3, 9, 2]
heapq.nlargest(3, arr)   # [9, 5, 3]
heapq.nsmallest(3, arr)  # [1, 2, 3]
‚ñ∂Max-Heap & Priority QueueWorkarounds and patternsO(log n)
# Max-heap: negate values
heapq.heappush(heap, -val)

# Priority Queue with heap
heap = []
heapq.heappush(heap, (priority, item))
priority, item = heapq.heappop(heap)
▶merge(*iterables, key, reverse)Merge multiple sorted inputs into oneO(n) total
# Merge pre-sorted iterables (lazy iterator)
a = [1, 3, 5]
b = [2, 4, 6]
list(heapq.merge(a, b))  # [1, 2, 3, 4, 5, 6]

# With key function
list(heapq.merge(
    [(1, 'a'), (3, 'c')],
    [(2, 'b')],
    key=lambda x: x[0]
))

queue Module

from queue import Queue, LifoQueue, PriorityQueue ‚Äî Thread-safe queue implementations.

from queue import Queue

from queue import PriorityQueue

from queue import Queue
q = Queue()

item = q.get()
# ... process item ...
q.task_done()  # signal task is complete

q.join()  # blocks until all tasks are done

Raises queue.Full or queue.Empty if not possible.

import queue
q = queue.Queue(maxsize=2)
